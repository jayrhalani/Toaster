# Toaster
Simple Toast Library

> Step 1. Add the JitPack repository to your build file

```build
allprojects {
  repositories {
		...
		maven { url 'https://jitpack.io' }
	}
}
 ``` 
  
  > Step 2. Add the dependency
  
 ```build
  dependencies {
	        implementation 'com.github.jayrhalani:Toaster:0.0.2'
	}
  ```
